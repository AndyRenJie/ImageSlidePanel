## ImageSlidePanel ##

图片照片墙的加载和滑动特效

Image_Slide_Panel图片照片墙的加载图片的时候一张一张叠加和加载完成后切换每一张的滑动特效，自定义ImageSlidePanel，按照初叶的使用习惯，
每一页应该是一个模板，所以此处命名时使用了Template关键字
